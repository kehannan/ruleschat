# Specification for the ASL Rules Assistant

## Overview

The ASL Rules Assistant is a tool that allows users to ask questions about the ASL rules and get answers from an AI assistant.

## Features

- Ask questions about the ASL rules
- Get answers from an AI assistant

## Current Implementation

The current implementation is a web application that allows users to ask questions about the ASL rules and get answers from an AI assistant.

## Future Implementation
