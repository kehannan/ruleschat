Accurate — but not cheap.

Fable can answer an {% if fable_fid %}easy eval{% else %}easy eval{% endif %} with 98.8% accuracy and a harder {% if fable_med %}medium eval{% else %}medium eval{% endif %} with 93.3% — scoring 100% on calculation questions in both — but at a cost of about 25–30¢ per query.

A more cost-effective model, {% if gpt54_fid %}GPT-5.4{% else %}GPT-5.4{% endif %}, has ~85% accuracy (90% recall, 80% calculation) at a cost of about 6¢ per query. There are cheaper and slower models with slightly worse performance.
