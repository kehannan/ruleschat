# Contributing

This is a personal project shared publicly for reference. Contributions are welcome, but please keep in mind the scope is intentionally narrow — this is an experiment in RAG-based rules lookup for a specific domain, not a general-purpose framework.

## What fits

- Bug fixes
- Mobile / UI improvements
- Eval tooling improvements
- Documentation corrections

## What probably doesn't fit

- New game support (the system prompt and vector store are ASL-specific)
- Auth rewrites or new user management features
- Significant architecture changes

## Getting started

See [README.md](README.md) for setup instructions.

## Submitting changes

1. Fork the repo and create a branch from `main`
2. Make your changes
3. Test locally — run `python tests/test_mobile.py` for UI tests
4. Open a pull request with a clear description of what changed and why

## Issues

If you find a bug or have a question, open an issue. Please include steps to reproduce for bugs.
